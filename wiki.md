# RotatingHeads2 Wiki

Welcome to the RotatingHeads2 Wiki. Here you can find all information about commands and usage.

## Commands

### Create Head
Creates a new rotating head at your current location.
**Usage:** `/rh create <name> [--center] [--name <player>] [--value <base64>]`
- `<name>`: Unique identifier for the head.
- `--center`: Centers the head on the block you are standing on.
- `--name <player>`: Uses the skin of the specified player.
- `--value <base64>`: Uses a custom texture value (base64 string).

**Examples:**
- `/rh create lobby_spawn --center`
- `/rh create notch_head --name Notch`
- `/rh create custom_head --value eyJ0ZXh0dXJlcyI6eyJTS0lOIjp7InVybCI6Imh0dHA6Ly90ZXh0dXJlcy5taW5lY3JhZnQubmV0L3RleHR1cmUv...`

### Edit Head
Edits the texture or properties of an existing head.
**Usage:** `/rh edit <head> [--name <player>] [--value <base64>] [--scale <value>]`
- `<head>`: The name of the head to edit.
- `--name <player>`: Updates the head to use the specified player's skin.
- `--value <base64>`: Updates the head to use the specified custom texture.
- `--scale <value>`: Updates the size of the head (default: 1.0). Note: Visual scaling requires a compatible ProtocolLib version.

**Examples:**
- `/rh edit lobby_spawn --name Akame`
- `/rh edit custom_head --scale 2.0`
- `/rh edit custom_head --value eyJ0ZXh0dXJlcyI6eyJTS0lOIjp7InVybCI6Imh0dHA6Ly90ZXh0dXJlcy5taW5lY3JhZnQubmV0L3RleHR1cmUv...`

### Move
Moves an existing head to specified coordinates.
**Usage:** `/rh move <head> <x> <y> <z>`
- `<head>`: The name of the head to move.
- `<x> <y> <z>`: The new coordinates. Supports `~` for current position.
  - Example: `/rh move my_head 100 64 100` (Moves head to 100 64 100)
  - Example: `/rh move my_head ~ 70 ~` (Moves head to Y=70, keeps X and Z)
  - Permission: `rh.move`

### Delete Head
Deletes an existing head.
**Usage:** `/rh delete <head> [--force]`
- `<head>`: The name of the head to delete.
- `--force`: Deletes the head immediately without asking for confirmation.

**examples:**
- `/rh delete lobby_spawn`
- `/rh delete lobby_spawn --force`

### Teleport
Teleports you or another player to a head.
**Usage:** `/rh teleport <head> [player] [--silent]`
- `<head>`: The name of the head.
- `[player]`: Optional. The player to teleport.
- `--silent`: Hides the teleport message.

### List Heads
Lists all rotating heads.
**Usage:** `/rh list`

### Move Here
Moves an existing head to your current location.
**Usage:** `/rh movehere <head> [--center]`

### Clone Head
Clones an existing head configuration to a new name.
**Usage:** `/rh clone <head> <new_name>`

### Reload
Reloads the plugin configuration, heads, and clears unused caches (memory/skin).
**Usage:** `/rh reload`

## Permissions
- `rh.create`: Create new heads.
- `rh.delete`: Delete heads.
- `rh.edit`: Edit heads.
- `rh.teleport`: Teleport to heads.
- `rh.list`: List heads.
- `rh.movehere`: Move heads.
- `rh.clone`: Clone heads.
- `rh.reload`: Reload plugin.
- `rh.convert`: Convert from old plugin versions.
